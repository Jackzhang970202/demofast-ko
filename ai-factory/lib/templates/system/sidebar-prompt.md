# 模板A：左侧菜单管理系统

## 适用场景
各类管理系统、业务平台、后台系统、CRM、ERP、OA 等需要数据管理和 CRUD 操作的系统。

## 配色系统
- 页面背景: #F0F4F8
- 侧边栏背景: #FFFFFF
- 主色 Primary: #3B82F6（蓝色）
- 主色深 Primary Dark: #1E40AF
- 文字主色: #1E293B
- 次要文字: #64748B
- 边框: #E2E8F0
- 悬浮背景: #F1F5F9
- 卡片背景: #FFFFFF
- 成功色: #16A34A
- 警告色: #CA8A04
- 错误色: #DC2626

## 整体布局结构
```
<div class="layout">                    <!-- display:flex; min-height:100vh -->
  <aside class="sidebar">               <!-- width:220px; flex-shrink:0; background:#fff; border-right:1px solid #e2e8f0 -->
    <div class="sidebar-header">        <!-- 系统标题区，底部边框分隔 -->
    <div class="sidebar-nav">           <!-- flex-1; overflow-y:auto -->
      <button class="nav-btn">首页</button>
      <div class="nav-section-header">业务模块</div>
      <button class="nav-btn">业务模块1</button>
      <button class="nav-btn">业务模块2</button>
      ...
      <div class="nav-section-header">通用模块</div>
      <button class="nav-btn">用户管理</button>

      ...
    <div class="sidebar-footer">        <!-- 底部操作按钮区 -->
  <main class="main-content">           <!-- flex-1; overflow-auto; padding:24px -->

```

## 侧边栏导航样式
- 宽度: 220px，固定不伸缩
- 按钮样式: 全宽、左对齐、padding 11px 20px、无边框背景
- 左侧 3px 边框作为激活状态指示器
- 激活状态: 背景 #EFF6FF、文字 #3B82F6、左边框 #3B82F6、加粗
- 悬浮状态: 背景 #F1F5F9
- 模块分隔: 顶部 1px 边框 + 小字号标题
- 过渡动画: 0.15s

## 通用模块（必须包含）
导航栏"通用模块"区域必须包含以下 6 个模块：

1. **用户管理** - 用户名、姓名、邮箱、手机号、角色（下拉选择）、部门（下拉选择）、状态（启用/停用）
2. **角色管理** - 角色名称、描述、权限数、成员数、状态（启用/停用）
3. **系统设置** - 配置项名称、配置值、说明、分类、状态（启用/停用）
4. **操作日志** - 时间、操作人、操作类型（下拉）、操作对象、IP、结果（成功/失败）
5. **文件管理** - 文件名、大小、类型（下拉）、上传者、上传时间、状态（正常/已归档）
6. **数据报表** - 报表名称、周期（下拉）、生成时间、操作人、状态（下拉）

## CRUD 页面样式
每个模块页面包含：
- **标题栏**: 模块名称 + 记录数 + 新增按钮（右对齐）
- **筛选栏**: 圆角卡片内，下拉筛选 + 文本搜索 + 重置按钮，flex-wrap 布局
- **数据表格**:
  - 表头: padding 12px 16px、背景 #F8FAFC、底部 2px 边框、小字号次要文字
  - 行: padding 11px 16px、底部 1px 分隔线、悬浮背景 #F8FAFC
  - 状态列: 使用彩色标签（启用=绿色bg、停用=红色bg）
  - 操作列: "编辑"（蓝色文字）+ "删除"（红色文字）
  - 空状态: 居中提示"暂无数据"
- **分页**: 底部居中，显示总记录数和每页数量

## 弹窗表单样式
- 遮罩层: 固定定位、半透明黑色背景、毛玻璃模糊
- 表单框: 白色背景、圆角 14px、内边距 28px、宽度 560px
- 表单项: 标签 + 输入框/下拉框/文本域
- 必填项标签用主色高亮
- 底部按钮: 确认（主色）+ 取消（边框按钮），flex 均分
- 点击遮罩层可关闭弹窗

## 删除确认弹窗
- 较小宽度 400px，居中
- 警告图标 + 确认标题 + 提示文字
- 确认（红色危险按钮）+ 取消（边框按钮）

## 状态标签样式
- 绿色（成功/启用/正常）: bg #F0FDF4, 文字 #16A34A
- 红色（失败/停用/异常）: bg #FEF2F2, 文字 #DC2626
- 黄色（进行中/待处理）: bg #FEFCE8, 文字 #CA8A04
- 蓝色（信息类）: bg #EFF6FF, 文字 #2563EB

## 统计卡片样式（用于首页）
- 白色背景、圆角 12px、阴影、1px 边框
- 内边距 20px 24px
- 小字号标签 + 大号数值 + 趋势指示器

## 图表卡片样式（用于首页）
- 白色背景、圆角 12px、阴影、1px 边框
- 标题区域 + 图表内容区域
- 使用 SVG 绘制简单图表（饼图、柱状图、趋势图）
